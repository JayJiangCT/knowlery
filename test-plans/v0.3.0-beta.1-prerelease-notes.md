# Knowlery 0.3.0-beta.1

This is an internal beta for Knowlery 0.3.0. It is intended for team testing through BRAT before the public 0.3.0 release.

## What changed

- Knowlery is now framed as a personal knowledge review space instead of a one-time setup utility.
- The dashboard has been reorganized into:
  - Today
  - This note
  - Weekly Review
  - Review Menu
  - System
- Today now focuses on the next useful knowledge maintenance move.
- This note lets testers review the current Obsidian note as the center of a small workflow.
- Review Menu presents natural language review moves instead of foregrounding raw source skills.
- Weekly Review generates a local Knowledge Review Atlas with:
  - English default view
  - Chinese language toggle
  - Knowledge Hexagon
  - Knowledge Timeline
  - Topic Constellation
  - Knowledge Extensions
  - Next Batch prompts
- Claudian handoff remains explicit, with copy fallback.

## How to install with BRAT

1. Install and enable `Obsidian42 - BRAT`.
2. Run `BRAT: Add a beta plugin for testing`.
3. Paste the Knowlery GitHub repository URL.
4. Choose the latest beta release.
5. Enable Knowlery from Community plugins.
6. Confirm the installed version is `0.3.0-beta.1`.

## What to test

- Can a new user explain what Knowlery does within one minute?
- Does Today provide one clear next action?
- Is This note useful when a Markdown note is open?
- Is Review Menu clearer than browsing raw skills?
- Does Weekly Review generate a useful Knowledge Review Atlas?
- Does the English/Chinese toggle work in `latest.html`?
- Does Claudian send/copy fallback behave correctly?
- Does System feel like maintenance rather than the main product?
